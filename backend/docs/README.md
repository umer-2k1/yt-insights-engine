# Backend Docs

This folder tracks backend implementation context for **YT Insight Engine**.

- Provide a shared understanding of backend architecture and delivery scope.
- Keep progress visible through concise, living markdown docs.
- Document what is completed, in progress, and pending.

## Documents

- `architecture.md` - backend components and data flow.
- `features.md` - backend/API capabilities.
- `how-it-works.md` - request-to-insight execution path.
- `progress-tracker.md` - todo/pending/completed status.
