# Frontend Docs

This folder tracks frontend build context for **YT Insight Engine**.

## Purpose

- Keep the team aligned on what is being built.
- Document what is completed, in progress, and pending.
- Keep implementation intent clear without over-documenting.

## Documents

- `features.md` - frontend capabilities and UI scope.
- `how-it-works.md` - user flow and page behavior.
- `progress-tracker.md` - todo/pending/completed status.

## UI Direction (Current)

- Dark mode only.
- Primary background: `#171a1c`.
- Primary accent: `#02e976`.
- Style: sleek, modern, high-contrast, recommendation-first.
