# Frontend Features

## Branded Product

- Application name: **YT Insight Engine**.
- Core message: turn channel analytics into clear next-content decisions.

## MVP UI Scope

- Primary surface: insight dashboard (core product experience).
- Landing page (supporting surface)
  - Hero with value proposition
  - Key features section
  - CTA section (`Analyze Channel`)
- Job status UI
  - `queued`, `running`, `completed`, `failed`
- Insight dashboard
  - Top performing themes
  - Fastest growing themes
  - Content gaps
  - Niche leader comparison
  - Suggested video ideas
- API integration
  - `POST /api/analyze-channel`
  - Polling `GET /api/analysis/:jobId`

## Visual/UX Goals

- Dark-only design.
- Green accent used for primary interaction and high-value metrics.
- Minimal and modern card system with clear information hierarchy.
